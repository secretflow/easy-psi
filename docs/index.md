# 什么是Easy PSI

Easy PSI 是隐语推出的一款独立白屏产品，可快速帮助用户体验隐语隐私求交（PSI）能力。<br />**Easy PSI 具有以下特点：**

1. 轻量化部署，基于 Docker 镜像，无需安装1。
2. 使用门槛低，交互式新手引导。
3. 安全，高效，值得信赖，生产级可用。

注：1. Easy PSI 0.2.0及之前的版本需要使用快速安装脚本。
